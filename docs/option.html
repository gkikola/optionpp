struct Option

Specifies a program option that the user has set from the command line.

The OptionParser class acts as a container for Options. Each option that the user has specified is stored, in order, within the OptionParser.

Members

char short_name;
Single-character short name for the option.
std::string long_name;
Long name for the option.
std::string argument;
The argument, if any, that was given with the option.
OptionDesc* desc;
Pointer to the OptionDesc object that describes this option.
